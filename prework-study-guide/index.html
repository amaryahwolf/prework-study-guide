HTML

  • HTML gives content structure
  • the head element contains information, or metadata, that helps the browser render the page correctly; the elements included are not visisble when a user looks at a webpage
  • the body element contains the content that is shown to the user; all the elements that we want the user to see and interact with must be placed inside this element
  • declarations are not case-sensitive; HTML elements are case-sensistive
  • an HTML attribute defines additional information about an element; e.g. html lang="en" means the we language for the webpage is english
  • there are lots of metadata tags to learn that u can use inside the head element
  • title defines the title for a webpage and is important for SEO
  • style defines the CSS styles associated with a website
  • base defines the base URL for a webpage
  • link connects an external resource to the HTML document
  • meta defines metadata such as the character set, description, keywords, author and viewport
  • script embeds client-side scripts in the HTML document
  • charset is the character set
  • !DOCTYPEhtml tells us the document is written in HTML
  • http-equiv sets an HTTP header for content
  • viewport sets the default view for our webpage
  • parent/child element - the body element is the parent and 3+ elements nestled inside the body is the child
  • semantic HTML elements: the name of the element describes its content and the role it plays on a webpage
  • div creates a division on the page
  • span creates an inline division
  • the opening tag and closing tag contain the contents of the visible HTML elements; closing tag has a forward slash
  • note: an element refers to the entire element from opening tag to closing tag; a tag refers only to what appears inside the angle brackets
  • h1, h2, h3, h4, h5, h6 are headings; u can only have one h1 on a webpage; this is important for screen readers
  • p is a paragraph or block of text
  • ul creates an unordered list
  • ol creates an ordered list
  • creates listed items
  • img contains information about images displayed on a webpage and can contain various attributes; e.g. src attribute defines the location of the image & alt attribute contains a text string that describes the appearance and functionality of the image - also important for screen readers and assistvie technologies
  • br creates a line of empty space or a line break between two blocks of content
  • a stands for anchor and creates links to the same webpage or other webpages; href attribute points to the URL for the link
  • section elements are semantic elements that define a section in a document that contains thematic content
